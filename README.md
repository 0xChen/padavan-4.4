# Padavan4.4固件编译

如需自己编译则点击Fork到您的仓库，然后点击Actions进入编译模式，我用的是小米R4A，那么你自己改成你的型号即可
如：原本文件是MI-R4A的，列如你是MI-4的设备，那就将MI-R4A编辑文件里面的所有写着MI-R4A的名称改成MI-4即可是小米4千兆版的编译
